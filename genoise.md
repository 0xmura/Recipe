# 基本のジェノワーズ

## 基本情報

ジェノワーズは全卵と砂糖を泡立てて得た気泡を熱で固定し、膨張剤に頼らず体積を出すスポンジ生地である。口当たりは軽いが、適切に焼き上げると保形性もあり、デコレーションケーキの土台として扱いやすい。家庭では15cm丸型一台分を基準にすると再現と比較がしやすい。作業記録を残すと改善点を追跡しやすい。この基準を固定する。

家庭再現では、卵液の温度、泡立ての到達点、粉と油脂の混合順を一定に保つことが品質安定の鍵となる。生地を過度に消泡させず焼成へつなげると、きめの細かさとしっとり感を両立しやすい。焼成後に十分な放冷と保湿を行うことで、翌日のスライス性と口溶けが整う。工程時間の統一も再現性向上に有効である。判断軸を統一する。

## 材料と道具

### 材料 15cm丸型1台分

| 材料 | 分量 | 備考 |
|---|---|---|
| 全卵 | 2個 | Mサイズ相当 |
| 上白糖 | 60g | グラニュー糖でも可 |
| 薄力粉 | 60g | 使用前にふるう |
| 無塩バター | 20g | 牛乳と合わせて溶かす |
| 牛乳 | 20g | バターと同時に保温 |

全卵と砂糖の比率は泡の安定と甘味の基準となるため、計量誤差を抑える必要がある。薄力粉はふるってダマを除去し、バターと牛乳は合わせて40℃前後で保温する。温度差を小さくしておくと生地へ戻す際の比重差が緩和され、油脂由来の沈み込みによる消泡を抑制しやすい。材料準備は焼成前に完了させる。投入順を守ること。

### 道具

| 道具 | 用途 |
|---|---|
| 15cm丸型 | 焼成型 |
| クッキングシート | 型への敷き込み |
| ボウル 2個 | 卵液と粉の混合 |
| ハンドミキサー | 起泡操作 |
| ゴムベラ | 粉合わせと最終混合 |
| 温度計 | 卵液と油脂液の温度管理 |

型には底と側面に紙を敷いて離型を安定させる。ボウルは湯せんに当てやすい金属製が扱いやすく、温度計は卵液40℃前後の確認に使う。ハンドミキサーは高速と低速を切り替えられる機種が望ましく、終盤を低速で整えることで粗大気泡を抑え、焼成後の空洞発生を減らしやすい。器具配置を固定すると作業が乱れにくい。計量器も準備する。

## 手順

### 生地作成

全卵と砂糖をボウルで合わせ、50℃前後の湯せんに当てながら混ぜ、生地温度を40℃前後まで上げる。湯せんを外したら高速で白くもったりするまで泡立て、最後に低速で全体を均一化する。生地を落とした筋がゆっくり消える状態を目安にし、泡立て不足と過泡立てを同時に避ける。湯せん時は湯が入らないよう注意する。縁の泡も均す。

薄力粉を再度ふるいながら加え、ゴムベラで底から返すように混ぜて粉気をなくす。次に保温したバター牛乳へ生地の一部を加えてなじませ、主生地へ戻して手早く均一化する。油脂液を直接入れるより消泡を抑えやすく、焼成後の層の乱れや底沈みを防ぎやすい。混合は最小回数で止める。混合後はすぐ型へ流し入れる。待機時間を作らない。

1. 全卵と砂糖を合わせ、湯せんで40℃前後まで温める。
2. 高速で泡立て、仕上げに低速で気泡を整える。
3. 薄力粉を加えて混ぜ、粉気が消えるまで合わせる。
4. バターと牛乳を生地の一部でのばし、主生地へ戻して混ぜる。

### 焼成と仕上げ

生地を型へ流し入れ、170℃に予熱したオーブンで約25分焼く。表面が均一なきつね色になり、中央を軽く押して戻る弾力があれば焼成完了の目安となる。焼成不足は冷却後の沈みにつながるため、色と弾力を同時に確認する。必要なら1〜3分単位で追加焼成して終点を合わせる。焼成中は扉の開閉を最小限にする。予熱不足も避ける。

焼き上がり直後に10cm程度の高さから一度落として蒸気を抜き、型から外して網で冷ます。粗熱が取れたら乾燥防止のため袋に入れ、最低半日休ませて水分分布を均一化する。休ませ後にスライスすると断面が崩れにくく、クリームを重ねる工程でも層の圧縮や欠けが起こりにくい。保管中は直射日光を避ける。室温変化も抑える。

1. 170℃で約25分焼き、色と弾力で焼成終点を確認する。
2. 焼成直後に一度落として蒸気を抜き、型から外して放冷する。
3. 粗熱後に袋へ入れ、最低半日休ませてから使用する。

## 原理

ジェノワーズの膨張は、起泡で形成した気泡核が加熱で膨らみ、卵タンパク質の熱変性で壁面が固定される過程で成立する。砂糖は液相の粘度と泡膜の安定性を高め、泡の合一を抑えて体積保持に寄与する。初期温度が不足すると泡立ち効率が低下し、同時間の攪拌でも到達体積が不足しやすい。適温管理が気泡密度を左右する。温度履歴を管理する。

薄力粉デンプンは加熱で糊化し、卵タンパク質の網目構造と重なってクラムの粘弾性を形成する。油脂と乳成分は保湿と口溶けを改善する一方、投入時期が早いと起泡を阻害しやすい。したがって温度管理、混合順、焼成終点、放冷後の保湿管理までを一連の工程として扱うことが品質再現に不可欠である。工程間の遅延は品質低下の要因になる。
