# 基本のジェノワーズ

## 基本情報

ジェノワーズは、全卵と砂糖をしっかり泡立てて生地の中に空気を抱き込み、その気泡をオーブンの熱で固定してふくらませるスポンジ生地である。ベーキングパウダーに頼らずに高さを出せるため、口当たりが軽く、デコレーションケーキの土台として使いやすい。家庭では15cm丸型1台分を基準にすると仕上がりを比較しやすい。

家庭で安定して焼くには、卵液の温度、泡立ての見極め、粉合わせの混ぜ方を毎回そろえることが大切である。泡をつぶし過ぎずに焼成へつなげると、きめが細かくしっとりした食感になりやすい。焼いたあとに十分冷ましてからラップで包んで休ませると、水分が落ち着いて翌日のスライスがしやすくなる。デコレーション時も形を保ちやすい。

## 材料と道具

### 材料 15cm丸型1台分

| 材料 | 分量 | 備考 |
|---|---|---|
| 全卵 | 2個 | Mサイズ相当 |
| 上白糖 | 60g | グラニュー糖でも可 |
| 薄力粉 | 60g | 使用前にふるう |
| 無塩バター | 20g | 牛乳と合わせて溶かす |
| 牛乳 | 20g | バターと同時に保温 |

全卵と砂糖の比率は、泡の立ち方と甘さのバランスを左右するため、分量はきちんと量り取る。薄力粉はふるってダマをなくし、バターと牛乳は合わせて40℃前後で温めておく。温度差が小さい状態で加えると生地になじみやすく、油脂が底へ沈んで泡が急に減るのを防ぎやすい。粉と液体は計量後にすぐ使える位置へ置く。並べて準備する。

### 道具

| 道具 | 用途 |
|---|---|
| 15cm丸型 | 焼成型 |
| クッキングシート | 型への敷き込み |
| ボウル 2個 | 卵液と粉の混合 |
| ハンドミキサー | 起泡操作 |
| ゴムベラ | 粉合わせと最終混合 |

型には底と側面に紙を敷いて、焼き上がり後にすぐ外せる状態をつくる。ボウルは湯せんに当てやすい金属製が使いやすく、温度計は卵液を40℃前後に保てているか確認するために使う。ハンドミキサーは高速と低速を切り替えられるものが扱いやすく、仕上げで低速にすると大きな気泡を整えやすい。紙はしわを伸ばして密着させる。

## 手順

### 生地作成

全卵と砂糖をボウルで合わせ、50℃前後の湯せんに当てながら混ぜて、卵液を40℃前後まで温める。湯せんを外したら高速で白くもったりするまで泡立て、最後に低速で全体の気泡を整える。生地を落とした筋がゆっくり消える状態になれば泡立ての目安になる。湯せんの湯は沸騰させず、鍋肌が静かに揺れる程度の火加減を保つ。

薄力粉を再度ふるいながら加え、ゴムベラで底から返すように混ぜて粉気をなくす。次に、温めたバター牛乳へ生地の一部を入れて先になじませ、元の生地へ戻して手早く均一にする。油脂液を直接入れるよりも泡を保ちやすく、焼いたときの層の乱れや底の詰まりを抑えやすい。混ぜ終えたらすぐ型へ流し入れる。表面の大きな泡だけ除く。

1. 全卵と砂糖を合わせ、湯せんで40℃前後まで温める。
2. 高速で泡立て、仕上げに低速で気泡を整える。
3. 薄力粉を加えて混ぜ、粉気が消えるまで合わせる。
4. バターと牛乳を生地の一部でのばし、元の生地へ戻して混ぜる。

### 焼成と仕上げ

生地を型へ流し入れ、170℃に予熱したオーブンで約25分焼く。予熱は表示温度だけでなく庫内温度が安定してから始めると焼きむらを抑えやすい。表面が均一なきつね色になり、中央を軽く押して戻る弾力があれば焼き上がりの目安になる。必要に応じて1〜3分ずつ追加で焼き、焼き始めから20分までは扉を開けない。型は中央段に置く。

焼き上がり直後に10cmほどの高さから一度落として蒸気を抜き、型から外して網で冷ます。粗熱が取れたら乾燥を防ぐためラップで包み、最低半日休ませて水分を全体になじませる。休ませたあとに切ると断面が崩れにくく、クリームを重ねる作業でも層がつぶれにくい。保管中は直射日光を避ける。高温多湿の場所も避ける。密着させて包む。

1. 170℃で約25分焼き、色と弾力で焼き上がりを確認する。
2. 焼成直後に一度落として蒸気を抜き、型から外して放冷する。
3. 粗熱後にラップで包み、最低半日休ませてから使用する。

## 原理

ジェノワーズの体積は、起泡で生じた多数の気泡核が加熱で膨張し、その周囲を卵タンパク質の熱変性で固めることで維持される。卵白ではオボトランスフェリンが約61℃付近、オボアルブミンが80℃台で変性し、卵黄タンパク質も加熱で粘度が上がる。温度上昇に伴って気泡膜が強化されるため、焼成中に形が保たれる。これが体積維持の基礎になる。

砂糖は甘味だけでなく、泡の安定化にも関与する。糖が水相に溶けると粘度が上がり、気泡同士の合一が起こりにくくなる。また、砂糖濃度が上がると卵タンパク質の凝固開始温度は上方へ移り、急激な硬化を抑えながら加熱を進められる。湯せんで卵液を40℃前後に整える操作は、起泡性と安定性を両立させるための温度管理の要点である。

薄力粉の主成分である小麦デンプンは、含水条件下で加熱されると結晶構造が崩れて糊化し、クラムの骨格形成に寄与する。一般に小麦デンプンは50℃台から膨潤が始まり、60℃台で粘度上昇が明確になる。ここに卵タンパク質の網目が重なることで、軽さを保ちながらも崩れにくい内部構造ができる。焼き不足ではこの固定が不十分になる。

バターと牛乳の脂質は口溶けと保湿性を高めるが、起泡の初期段階で混入すると界面に入り込み、タンパク質の膜形成を妨げやすい。そこで起泡後半に加えることで、体積を保ちながら口当たりを整えやすくなる。焼成後にラップで包んで休ませる工程は、水分活性の偏りを小さくして組織を均一化し、切り分け時の割れやボソつきを抑える効果がある。
