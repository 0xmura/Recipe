# 基本のカヌレ

## 基本情報

カヌレはフランスのボルドー地方を発祥とする伝統的な焼き菓子であり、正式名称はカヌレ・ド・ボルドーという。16世紀ごろにボルドーの修道院で生まれたとされており、当時ワイン造りで清澄剤として使用した後に大量に残った卵黄の活用法として考案されたという説が有力である。外側は濃い焦げ茶色のカラメリゼされたクラストが形成され、内側はしっとりとした柔らかいクラムになるという、二層の食感の対比がこの菓子の特徴である。カラメルの香ばしい風味にバニラとラム酒の芳醇な香りが溶け合い、独特の風味をつくりだす。

このレシピは家庭用の分量で、伝統的な銅製カヌレ型を使用して6個分を想定している。銅製型のサイズは直径5.5cm・高さ4.5cm程度のものを用いる。銅は熱伝導率が非常に高いため型全体が均一に加熱され、特徴的なクラストを均一に形成する素材である。生地は前日に仕込んで一晩休ませることがきめ細かな食感を得るうえで欠かせない工程であり、みつろうをコーティング剤として使用することで銅型との相性が高まりよいクラストが得られる。

## 材料と道具

### 材料 6個分

| 材料 | 分量 |
|---|---|
| 牛乳 | 160ml |
| バニラビーンズ | 1本 |
| 砂糖 | 80g |
| T45小麦粉 | 35g |
| 卵黄 | 2個 |
| 無塩バター | 16g |
| ラム | 8g |
| コアントロー | 8g |
| みつろう | 15〜20g |

牛乳は乳脂肪分3.5%以上の全脂牛乳を160ml用意する。全脂牛乳を使うことでクラムにコクと滑らかさが生まれ、スキムミルクや低脂肪乳では同等の仕上がりを得ることが難しい。バニラビーンズはマダガスカル産またはタヒチ産の太くて香り高いさやを1本使用する。砂糖は上白糖またはグラニュー糖を80g計量し、小麦粉はふるいにかけた状態で35g用意する。卵黄は2個使用し、無塩バターは16g準備する。ラムは8g、コアントローは8g用意する。みつろうは食品グレードの精製みつろうを15〜20g用意する。みつろうは型のコーティングに使用し、離型性と均一なカラメル化を実現する材料である。

厚手の鍋は牛乳を均一に温めるために使用し、薄手の鍋では焦げつきやすいため避ける。ボウルは生地を混ぜるために2つ用意し、一方は粉類用、もう一方は卵と牛乳を合わせるために使う。泡立て器は粉類と卵・牛乳を均一に混ぜ合わせるために必要で、空気を抱き込まないよう静かに混ぜる。細かい目のザルまたは濾し器は生地を漉して滑らかにするために使用し、この工程を怠るとクラムの食感にムラが生じる。温度計はバニラのアンフューゼ工程での正確な温度管理に使い、デジタル式のものが精度の面で優れている。みつろうの湯煎溶解には小鍋と湯煎用の鍋を1つずつ別途用意する。

## 手順

### アパレイユ

バニラのアンフューゼとは、バニラの香気成分を牛乳に溶出させる工程であり、完成品の香りの出来を左右する工程のひとつである。バニラのさやを縦に切り開き、ナイフの背でバニラシードをこそぎ取る。種とさやの両方を牛乳の中に入れ中火にかける。温度計で70〜75℃に達したら火を止める。この温度帯で加熱することで、高温加熱中に生じるバニリンをはじめとする揮発性の芳香成分の蒸気移行による損失を抑え、香気を十分に引き出す方法として知られている。鍋に蓋をしてそのまま30分以上おき、バニラの香りをじっくりと牛乳に移す。蓋をすることで揮発成分が逃げるのを防ぎ、香りの損失を最小限に抑えることができる。アンフューゼが終わったらバターを加えて余熱で溶かし、バニラのさやはゴムベラで軽く絞るようにして香りを出し切ってから取り除く。

ボウルに砂糖と小麦粉を入れ、泡立て器でよく混ぜ合わせる。別のボウルに卵黄を溶きほぐし、粉類のボウルに加えてゴムベラで均一になるまで混ぜる。ここへ人肌程度に冷ました牛乳を50ml程度ずつ少量ずつ加えながら、その都度なめらかになるように丁寧にすり混ぜていく。一度に加えると粉がダマになりやすく後から修正が難しいため、少量ずつ段階的に加えることが大切である。生地全体が均一になったら細かいザルで漉し、ラムとコアントローを加えてよく混ぜる。生地をラップで覆い冷蔵庫で一晩休ませる。この休ませる工程によってグルテンが落ち着き、バニラの香りがさらに生地全体にゆっくりと浸透し、焼き上がりの食感と風味が向上する。

1. バニラのさやを縦に切り開き、ナイフの背でバニラシードをこそぎ取る。種とさやをともに牛乳に入れ中火にかける。
2. 温度計で70〜75℃に達したら火を止め、蓋をして30分以上おく。
3. バターを加えて余熱で溶かし、バニラのさやをゴムベラで軽く絞って香りを出し切り取り除く。
4. ボウルに砂糖と小麦粉を入れ泡立て器で混ぜ合わせる。別のボウルで溶きほぐした卵黄を加えてゴムベラで均一に混ぜる。
5. 人肌程度に冷ました牛乳を50ml程度ずつ加えながら、その都度なめらかになるようにすり混ぜる。
6. 細かいザルで漉し、ラムとコアントローを加えて混ぜる。ラップで覆い冷蔵庫で一晩休ませる。

### 焼成

生地は冷蔵庫で十分に冷えた状態のまま使用する。オーブンを230℃の上下火で予熱し、予熱完了後に生地を型へ注いですぐ焼成に入る。冷たい生地を高温で立ち上げることで、外側のクラスト形成と内側の火入れのバランスを取りやすくする。

銅製型のみつろうコーティングは、カヌレのクラスト形成に関わる工程のひとつである。まず、空の銅製型をオーブンに入れ200℃で5〜10分間予熱する。型を十分に熱しておくことで、みつろうが型内面に均一に広がりやすくなる。次に、みつろうを湯煎にかけて完全に液状になるまでゆっくりと溶かす。湯煎の湯温は80℃程度を目安とし、みつろうの温度が65〜70℃になるよう管理する。直火で溶かすと局所的に過熱して発煙するおそれがあるが、湯煎を用いることで安全に均一な溶解が実現できる。オーブンから取り出した熱い型に液状のみつろうをすばやく注ぎ入れ、型の縁いっぱいまで満たす。そのまま型を傾けながら数秒で内面全体——溝、側面、底——にみつろうが行き渡るように回転させ、余分なみつろうを小鍋に戻す。型を逆さにしてペーパータオルの上に置き、余分なみつろうを完全に切る。その後、型を正立させ内面に薄く均一なみつろう層が残った状態で冷蔵庫に入れ、みつろうが完全に固まるまで10〜15分間冷やす。この流し込みから戻す方法により、刷毛では届きにくい溝の細部まで均一にコーティングでき、焼成中のカラメル化が均一になる。

みつろうが固まったコーティング済みの型に、一晩休ませた生地を軽く混ぜてから型の8分目まで静かに注ぐ。生地を注ぐ際は泡立てないよう注意し、泡が生じた場合は竹串で潰してから焼成に移ること。型をオーブンに入れ230℃で10分焼く。その後、温度を190℃に下げさらに60分焼く。焼き上がりの目安は外側が濃い焦げ茶色でほぼ黒に近い色になっていることであり、色が薄いうちは焼き不足であるためしっかりと色づくまで焼く。

焼き上がったらすぐにオーブンから取り出し、1分ほど待ってから型から外して網の上に並べ完全に冷ます。熱いうちは内側が柔らかすぎて形が崩れやすいため、少なくとも30分は冷ましてから食べることで外はパリッと内はしっとりした食感を楽しめる。焼いた当日はクラストのパリパリ感があり、翌日以降はクラストが少しずつ柔らかくなっていくため、できる限り当日中に食べると美味しい。

1. 生地は冷蔵庫で冷たい状態のまま使う。オーブンを230℃の上下火で予熱する。
2. 銅製型をオーブンに入れ5〜10分間予熱する。
3. みつろうを湯煎にかけ、湯温80℃・みつろう温度65〜70℃を目安に完全に液状になるまで溶かす。
4. オーブンから取り出した型にみつろうを縁いっぱいまで注ぎ、型を傾けながら内面全体に行き渡らせた後、余分を小鍋に戻す。
5. 型を逆さにしてペーパータオルの上に置き余分なみつろうを切る。型を正立させ冷蔵庫で10〜15分冷やして固める。
6. 一晩休ませた生地を軽く混ぜ、型の8分目まで静かに注ぐ。泡が生じた場合は竹串で潰す。
7. 230℃で10分焼き、190℃に下げてさらに60分焼く。外側が濃い焦げ茶色になるまで焼く。
8. 焼き上がったらすぐにオーブンから取り出し、1分ほど待ってから型から外して網の上に並べ完全に冷ます。

## 原理

カヌレのクラストとクラムの食感の差異は、高温焼成と生地の配合が組み合わさることで生まれる。外側の黒いクラストは、メイラード反応とカラメル化反応の複合によって形成される。メイラード反応とは、アミノ酸の遊離アミノ基と還元糖のカルボニル基が高温下で縮合を起こし、メラノイジンと呼ばれる褐色色素と複数の芳香族化合物を生成する非酵素的褐変反応である。この反応は主に150℃以上で顕著に進み、焼成初期の230℃という高温がその進行を促す。カラメル化反応はこれとは独立した反応であり、ショ糖が160〜180℃で熱分解してフルフラールやヒドロキシメチルフルフラールなどのフラン系化合物のほか、各種のピラン系化合物を生成する。これら二つの反応が同時進行することで、苦みと香ばしさを持つ複雑な風味と硬く薄いクラストが生まれる。

生地を一晩休ませる工程には、グルテンの緩和と成分の均一化という二つの化学的意義がある。グルテンはグリアジンとグルテニンという二種のタンパク質が水の存在下で水素結合と疎水性相互作用によって結びつき、三次元ネットワーク構造を形成したものである。混合直後の生地はこのネットワークが強く引っ張られた状態にあり、そのまま加熱すると硬い食感になる。冷蔵状態での長時間静置によってタンパク質分子の再配列が緩やかに進み、グルテン結合が弛緩して焼き上がりのクラムが滑らかでしっとりした仕上がりになる。また、バニラの香気成分であるバニリンやp-ヒドロキシベンズアルデヒドなどのフェノール性化合物は、冷蔵環境下でも拡散係数に従って生地全体にゆっくりと移行し、均一な香りの分布が形成される。

デンプンの糊化はクラムの粘弾性構造を生む化学変化である。小麦粉に含まれるデンプンは、直鎖状のアミロースと枝分かれ構造のアミロペクチンから構成されており、常温では水と接触しても結晶性のラメラ構造を維持する。オーブン内で温度が60〜70℃に達すると、デンプン粒が急速に水分を吸収して膨潤し、アミロース分子が溶出してゲル状の連続相を形成する。これがデンプンの糊化であり、この変化によってクラムは粘弾性のあるもちっとした構造へと変わる。一方、外表面は230℃という高温に直接さらされるため水分が急速に蒸発し、糊化したゲルが脱水・硬化してクラストを形成する。内外での温度勾配と水分状態の差異がクラストとクラムの明確な食感差を生む原理である。

みつろうは離型剤としての働きに加え、焼成時の熱移動を調整する層としても機能する。みつろうの主成分はミリシルパルミテートなどの長鎖脂肪酸エステルであり、融点は約62〜65℃である。焼成開始後に型の熱でみつろうが溶け出すと、生地と銅型の界面に薄い液体膜が形成される。この膜は型から生地への熱伝導を若干緩衝し、クラスト形成の均一性を高める効果をもたらす。バターは不飽和脂肪酸やタンパク質を含むため焼成中に酸化・変性が起きやすく、みつろうと比較して安定したコーティング層の維持が難しい。伝統的な銅型を使うレシピではみつろうが用いられてきた背景には、こうした熱的・化学的安定性の違いがある。

ラムに含まれるエタノールは沸点が約78℃であり、焼成開始後に水よりも早く気化する。このエタノールの急速な気化によって生地内部から多数の気泡が発生し、クラムに特徴的な多孔質構造が形成される。ラムやコアントロー特有のエステル系芳香成分はエタノールが揮発した後も加熱分解物として一部が残留し、焼成後の香りに寄与する。バニラビーンズの香気成分であるバニリンはフェノール性アルデヒドであり、脂質に対して高い親和性を持つため牛乳の乳脂肪に溶け込みやすい。70〜75℃での低温アンフューゼによって水溶性と脂溶性の両成分を牛乳に均一に溶出させ、焼成後もバニリンを中心とした芳香が持続する仕組みになっている。
