# 基本のカヌレ

## 基本情報

カヌレはフランスのボルドー地方を発祥とする伝統的な焼き菓子であり、正式名称はカヌレ・ド・ボルドーという。16世紀ごろにボルドーの修道院で生まれたとされており、当時ワイン造りで清澄剤として使用した後に大量に残った卵黄の活用法として考案されたという説が有力である。外側は濃い焦げ茶色のカラメリゼされたクラストが形成され、内側はしっとりとした柔らかいクラムになるという、二層の食感の対比が最大の特徴である。カラメルの香ばしい風味にバニラとラム酒の芳醇な香りが溶け合い、他に類を見ない独特の風味をつくりだす。

このレシピは家庭用の分量で、伝統的な銅製カヌレ型を使用して6個分を想定している。銅製型のサイズは直径5.5cm・高さ4.5cm程度のものを用いる。銅は熱伝導率が非常に高いため型全体が均一に加熱され、特徴的なクラストの均一な形成に不可欠な素材である。生地は前日に仕込んで一晩休ませることがきめ細かな食感を得るうえで欠かせない工程であり、みつろうをコーティング剤として使用することで銅型との相性が高まり理想的なクラストが得られる。

## 材料と道具

| バニラビーンズ | 1/2本 | マダガスカル産またはタヒチ産推奨 |
| 砂糖 | 110g | 上白糖またはグラニュー糖 |
| 薄力粉 | 50g | ふるいにかけておく |
| 全卵 | 1個 | |
| 卵黄 | 1個 | |
| 無塩バター | 25g | |
| ダークラム | 25ml | アグリコールラム推奨 |
| みつろう | 15〜20g | 食品グレードの精製みつろう |

牛乳は乳脂肪分3.5%以上の全脂牛乳を250ml用意する。全脂牛乳を使うことでクラムにコクと滑らかさが生まれ、スキムミルクや低脂肪乳では同等の仕上がりを得ることが難しい。バニラビーンズはマダガスカル産またはタヒチ産の太くて香り高いさやを半本使用する。砂糖は上白糖またはグラニュー糖を110g計量し、薄力粉はふるいにかけた状態で50g用意する。全卵1個と卵黄1個を合わせて使用し、無塩バターは25g準備する。ラム酒はダークラムを25ml用意し、アグリコールラムを選ぶとさとうきびの豊かな風味がより強く引き出される。みつろうは食品グレードの精製みつろうを15〜20g用意する。みつろうは型のコーティングに使用し、離型性と均一なカラメル化を実現するために不可欠な材料である。

厚手の鍋は牛乳を均一に温めるために使用し、薄手の鍋では焦げつきやすいため避けるのが望ましい。ボウルは生地を混ぜるために2つ用意し、一方は粉類用、もう一方は卵と牛乳を合わせるために使う。泡立て器はドライ材料とウェット材料を均一に混ぜ合わせるために必要で、空気を抱き込まないよう静かに混ぜることが重要である。細かい目のザルまたは濾し器は生地を漉して滑らかにするために使用し、この工程を怠るとクラムの食感にムラが生じる。温度計はバニラのアンフューゼ工程での正確な温度管理に不可欠であり、デジタル式のものが精度の面で優れている。みつろうの湯煎溶解には小鍋と湯煎用の鍋を1つずつ別途用意する。

## 手順

### 前日の仕込み

バニラのアンフューゼとは、バニラの香気成分を牛乳に溶出させる工程であり、完成品の香りの出来を左右する最も重要な工程のひとつである。バニラのさやを縦に切り開き、ナイフの背でバニラシードをこそぎ取る。種とさやの両方を牛乳の中に入れ中火にかける。温度計で70〜75℃に達したら火を止める。この温度帯での加熱が重要で、100℃近くまで加熱するとバニリンをはじめとする揮発性の芳香成分が蒸散して香りが失われるため、低温での抽出が香気を最大限に引き出す方法として知られている。鍋に蓋をしてそのまま30分以上おき、バニラの香りをじっくりと牛乳に移す。蓋をすることで揮発成分が逃げるのを防ぎ、香りの損失を最小限に抑えることができる。アンフューゼが終わったらバターを加えて余熱で溶かし、バニラのさやはゴムベラで軽く絞るようにして香りを出し切ってから取り除く。

ボウルに砂糖と薄力粉を入れ、泡立て器でよく混ぜ合わせる。別のボウルに全卵と卵黄を溶きほぐし、粉類のボウルに加えてゴムベラで均一になるまで混ぜる。ここへ人肌程度に冷ました牛乳を50ml程度ずつ少量ずつ加えながら、その都度なめらかになるように丁寧にすり混ぜていく。一度に加えると粉がダマになりやすく後から修正が難しいため、少量ずつ段階的に加えることが大切である。生地全体が均一になったら細かいザルで漉し、ラム酒を加えてよく混ぜる。生地をラップで覆い冷蔵庫で一晩休ませる。この休ませる工程によってグルテンが落ち着き、バニラの香りがさらに生地全体にゆっくりと浸透し、焼き上がりの食感と風味が格段に向上する。

### 焼成当日

焼く1時間前に生地を冷蔵庫から取り出し、室温に戻しておく。オーブンを230℃の上下火で予熱する。生地が冷たいままだと型に注いだ後に急激な温度差が生じ、クラストの均一な形成が妨げられる場合があるため、この室温に戻す工程を省略してはならない。予熱が十分に完了したオーブンで焼くことで、最初の高温段階での急速なクラスト形成が適切に進む。

銅製型のみつろうコーティングは、カヌレのクラスト形成において最も重要な工程のひとつである。まず、空の銅製型をオーブンに入れ200℃で5〜10分間予熱する。型を十分に熱しておくことで、みつろうが型内面に均一に広がりやすくなる。次に、みつろうを湯煎にかけて完全に液状になるまでゆっくりと溶かす。湯煎の湯温は80℃程度を目安とし、みつろうの温度が65〜70℃になるよう管理する。直火で溶かすと局所的に過熱して発煙するおそれがあるが、湯煎を用いることで安全に均一な溶解が実現できる。オーブンから取り出した熱い型に液状のみつろうをすばやく注ぎ入れ、型の縁いっぱいまで満たす。そのまま型を傾けながら数秒で内面全体——溝、側面、底——にみつろうが行き渡るように回転させ、余分なみつろうを小鍋に戻す。型を逆さにしてペーパータオルの上に置き、余分なみつろうを完全に切る。その後、型を正立させ内面に薄く均一なみつろう層が残った状態で冷蔵庫に入れ、みつろうが完全に固まるまで10〜15分間冷やす。この流し込みから戻す方法により、刷毛では届きにくい溝の細部まで均一にコーティングでき、焼成中のカラメル化が均一になる。

みつろうが固まったコーティング済みの型に、一晩休ませた生地を軽く混ぜてから型の8分目まで静かに注ぐ。生地を注ぐ際は泡立てないよう注意し、泡が生じた場合は竹串で潰してから焼成に移ることが望ましい。型をオーブンに入れ230℃で15分焼く。その後、温度を180℃に下げさらに60〜70分焼く。焼き上がりの目安は外側が濃い焦げ茶色でほぼ黒に近い色になっていることであり、色が薄いうちは焼き不足であるためしっかりと色づくまで焼くことが重要である。

オーブンから取り出したら、すぐに型から外して網の上に並べ完全に冷ます。熱いうちは内側が柔らかすぎて形が崩れやすいため、少なくとも30分は冷ましてから食べることで外はパリッと内はしっとりという理想的な食感を楽しめる。焼いた当日が最もクラストのパリパリ感が際立っており、翌日以降はクラストが少しずつ柔らかくなっていくため、できる限り当日中に食べることが推奨される。

## 原理

カヌレの最大の特徴であるクラストとクラムの食感の差異は、高温焼成と生地の配合が組み合わさることで生まれる。外側の黒いクラストは、砂糖が高温にさらされることで起こるカラメル化反応と、糖とアミノ酸が反応するメイラード反応の複合によって形成される。これらの反応が進むことで、苦みと香ばしさを併せ持つ独特の風味と、硬く薄いクラストが生まれ、カヌレ特有の外観と食感をつくりだす。

生地を一晩休ませる理由は、グルテンの緩和と食材の均一化にある。混合直後の生地はグルテンが強く形成されており、そのまま焼くと固い食感になってしまう。冷蔵庫で一晩休ませることでグルテンが弛緩し、焼き上がりのクラムが滑らかでしっとりした仕上がりになる。また、冷蔵状態でもバニラの香気成分が生地全体にゆっくりと拡散し、風味の均一化にも寄与する。

焼成中にはデンプンの糊化も重要な役割を果たす。薄力粉中のデンプン粒は常温では水と混合しても構造を保っているが、オーブン内で60〜70℃に達すると急速に水分を吸収して膨潤し、粒が崩壊してデンプンがゲル状に変化する。この現象をデンプンの糊化といい、糊化によってクラムは粘弾性のあるもちっとした構造へと変わる。一方、外側は高温により速やかに乾燥・硬化するため、内部の糊化ゲルが閉じ込められることなく外のクラストとの明確な食感差が生まれる。

みつろうは離型剤としての役割だけでなく、焼成中に型とクラストの間に薄い断熱・潤滑層をつくり、熱を緩衝することでカラメル化をより均一に促進させる効果がある。みつろうの融点は約62〜65℃であり、焼成開始直後に溶け出してカヌレの底と側面に広がり、生地とみつろうが一体となってクラストが形成される。バターでは同等の均一なクラストを得ることが難しく、伝統的な銅型を使うレシピではみつろうが不可欠とされている。

ラム酒は単なる香りづけにとどまらず、アルコールの揮発性によって焼成中に生地の内部から気泡を発生させ、クラムの多孔質な構造を形成するのを助ける役割も持つ。アルコールの沸点は約78℃であるため、焼成中に水よりも先に急激に気化し、生地内部に特徴的な気泡の痕跡を残す。バニラビーンズは香りの核となるが、牛乳を低温でじっくりと加熱して蓋をした状態で香りを移すことで、焼成後も芳醇な香りが持続する。
