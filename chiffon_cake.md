# 基本のシフォンケーキ

## 基本情報

シフォンケーキは、卵白を泡立てたメレンゲを生地に折り込み、型に流して焼くことで、きめ細かくふんわりとした軽い食感を生み出すケーキである。植物油を使用するため、バターを用いるスポンジケーキよりもしっとりした口当たりになるのが特徴で、焼成中に型の内壁を伝って高く立ち上がる独特のシルエットが生まれる仕組みをもつ。

本レシピは家庭用の分量として、17cmシフォン型1台分を基準としている。卵黄生地とメレンゲをそれぞれ別々に仕込み、それぞれを丁寧に仕上げてから合わせることが成功の鍵である。風味付けにはバニラビーンズ、オレンジリキュール、キルシュを組み合わせることで、複雑で奥行きのある豊かな香りに仕上げることができる。

## 材料・道具

### 材料（17cmシフォン型1台分）

#### 卵黄生地
| 材料 | 分量 |
|------|------|
| 薄力粉 | 120g |
| ベーキングパウダー | 3g |
| 卵黄 | 100g |
| はちみつ（卵黄用） | 10g |
| 太白ごま油 | 80g |
| 牛乳 | 120g |
| バニラビーンズ | 1/2本 |
| オレンジリキュール | 少量 |
| キルシュ | 少量 |

#### メレンゲ
| 材料 | 分量 |
|------|------|
| 卵白 | 200g |
| グラニュー糖（卵白用） | 100g |
| 乾燥卵白 | 5g |

### 道具

シフォン型（17cm）、ハンドミキサーまたはスタンドミキサー、大ボウル2個、ゴムべら、泡立て器、粉ふるい、計量スケール、オーブンが必要である。シフォン型は底が外れるタイプかアルミ製のものを使用する。型にはバターや油を塗らないことが重要で、生地が型の壁をつかんで上に伸びることで、シフォンケーキ特有の高さが生まれる。

## 手順

### 準備

オーブンを170℃に予熱しておく。薄力粉とベーキングパウダーを合わせてよくふるっておく。バニラビーンズはさやに縦に切り込みを入れ、ナイフの背で種をていねいにこそいで取り出す。卵黄と卵白はあらかじめ分けておき、メレンゲ用の卵白は泡立てる直前まで冷蔵庫でしっかり冷やしておくと、泡立ちが安定しやすくなる。

### 卵黄生地を作る

ボウルに卵黄とはちみつを入れ、泡立て器でよく混ぜ合わせる。太白ごま油を少しずつ加えながらその都度しっかり乳化させるように混ぜ、続いて牛乳、バニラビーンズの種、オレンジリキュール、キルシュを加えてよく混ぜる。最後にふるっておいた粉類を加え、ダマがなくなるまで丁寧に均一に混ぜて滑らかな卵黄生地を仕上げる。

### メレンゲを作る

冷やした卵白にグラニュー糖と乾燥卵白を合わせたものを3回に分けて少しずつ加え、ハンドミキサーで泡立てる。最初は低速で卵白全体をていねいにほぐし、グラニュー糖の1/3量を加えたあとは中〜高速で力強く泡立て続ける。ツノが立ち、なめらかで光沢があり、やや硬めのしっかりとした状態になったらメレンゲの完成である。

### 生地を合わせる

卵黄生地のボウルにまずメレンゲの1/3量を加え、泡立て器で均一になるまで全体をよく混ぜてしっかりとなじませる。この工程でメレンゲの泡が多少つぶれても構わない。続いて残りのメレンゲを2回に分けて加えて、ゴムべらで底から大きくすくうように折り込みながら、泡をなるべくつぶさないよう静かに丁寧に混ぜ合わせる。

### 焼成と仕上げ

型に生地をゆっくりと流し込み、台の上に数回打ちつけて内部の大きな気泡を抜く。170℃のオーブンで約40〜45分かけてじっくり焼く。竹串を中心部に刺して何もついてこなければ焼き上がりの目安である。焼き上がったらすぐに型を逆さにして粗熱を取りながら完全に冷まし、十分に冷めてから型の側面にナイフを差し込んで取り出す。

## 原理

シフォンケーキの軽い食感は、メレンゲに含まれる無数の気泡によって生み出される。卵白タンパク質は激しく泡立てることによって変性し、気泡を安定的に包む薄い膜を形成する。焼成中にこの気泡が熱膨張することで生地が大きくふくらみ、タンパク質と小麦粉のグルテンが熱によって固まることで、その軽い構造が焼成後も維持される。

バターではなく太白ごま油などの植物油を使う理由は、油の流動性にある。バターは常温で固体であるため、冷えると生地の食感をかたくしてしまい、保存性にも影響が出る。一方、植物油は冷えても液体の状態を保つため、焼き上がりのしっとり感が冷蔵後も長く持続する。この特性こそが、シフォンケーキ独自のきめ細かくしっとりした食感を実現している。

乾燥卵白をメレンゲに加えるのは、メレンゲの安定性を高めるためである。乾燥卵白に含まれる卵白タンパク質が、泡の安定に寄与する。また、はちみつは転化糖を含み、砂糖よりも保水性が高いことから、焼き上がり後も生地のしっとり感を保つ効果がある。バニラ、オレンジリキュール、キルシュは風味の複雑さを与える香味素材として機能する。
