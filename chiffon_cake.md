# 基本のシフォンケーキ

## 基本情報

シフォンケーキは、卵白を泡立てたメレンゲを卵黄生地に折り込み、型に流して焼くことで、きめ細かくふんわりとした軽い食感を生み出すケーキである。植物油を使用するため、バターを用いるスポンジケーキよりもしっとりとした口当たりになるのが特徴で、焼成中に型の内壁を伝って高く立ち上がる独特のシルエットが生まれる仕組みをもつ。

本レシピは家庭用の分量として、17cmシフォン型1台分を基準としている。卵黄生地とメレンゲをそれぞれ別々に仕込み、丁寧に仕上げてから折り合わせることが成功の鍵であり、工程を順序よく進めることが均一な焼き上がりにつながる。風味付けにはバニラビーンズ、オレンジリキュール、キルシュを組み合わせることで、複雑で奥行きのある豊かな香りを引き出すことができる。

## 手順

### 準備

オーブンを170℃に予熱しておく。薄力粉とベーキングパウダーを合わせてよくふるっておく。バニラビーンズはさやに縦に切り込みを入れ、ナイフの背で種をていねいにこそいで取り出す。卵黄と卵白はあらかじめ分けておき、メレンゲ用の卵白は泡立てる直前まで冷蔵庫でしっかり冷やしておくと、泡立ちが安定しやすくなる。

1. オーブンを170℃に予熱する。
2. 薄力粉とベーキングパウダーを合わせてふるう。
3. バニラビーンズのさやに縦に切り込みを入れ、ナイフの背で種をこそいで取り出す。
4. 卵黄と卵白をあらかじめ分け、卵白は泡立てる直前まで冷蔵庫で冷やしておく。

### 卵黄生地を作る

ボウルに卵黄とはちみつを入れ、泡立て器でよく混ぜ合わせる。材料はあらかじめ常温に戻しておくと乳化しやすい。太白ごま油を少しずつ加えながらその都度しっかり乳化させてから、牛乳、バニラビーンズの種、オレンジリキュール、キルシュを加えてよく混ぜる。最後にふるった粉類を加え、粉気がなくなるまで手早く均一に混ぜる。粉を加えた後の過剰な混合はグルテンを発達させてかたい食感の原因となるため、つやが出た段階で混ぜるのをやめる。

1. ボウルに卵黄とはちみつを入れ、泡立て器でよく混ぜ合わせる。
2. 太白ごま油を少しずつ加えながら乳化させるように混ぜる。
3. 牛乳、バニラビーンズの種、オレンジリキュール、キルシュを加えてよく混ぜる。
4. ふるっておいた粉類を加え、粉気がなくなるまで手早く均一に混ぜる。

### メレンゲを作る

卵白を泡立てる前にボウルと泡立て器に油分や水分が残っていないことを確認する。わずかな油脂でも気泡膜の形成が妨げられ、安定したメレンゲにならないためである。グラニュー糖と乾燥卵白を合わせたものを3回に分けて加えながらハンドミキサーで泡立て、最初は低速でほぐしてから中〜高速に切り替える。ツノが立ちなめらかで光沢があるやや硬めの状態が仕上がりの目安で、泡立てすぎると気泡が粗くなりもろい組織になる。

1. ボウルと泡立て器に油分・水分がついていないことを確認する。
2. 冷やした卵白をボウルに入れ、低速でほぐす。
3. グラニュー糖と乾燥卵白を合わせたものを3回に分けて加え、中〜高速で泡立てる。
4. ツノが立ち、なめらかで光沢があり、やや硬めの状態になったら完成。

### 生地を合わせる

卵黄生地のボウルにまずメレンゲの1/3量を加え、泡立て器でためらわずによく混ぜてなじませる。この工程によって生地の比重をメレンゲに近づけ、残りの折り込みが均一になる下地をつくる。続いて残りのメレンゲを2回に分けて加え、ゴムべらでボウルの底から大きくすくい上げて反転させる折り込み動作を繰り返す。全体が均一になったら直ちに型へ注ぐ。混ぜすぎると気泡が消えてボリュームが失われるため、均一になった段階で止めることが重要である。

1. 卵黄生地にメレンゲの1/3量を加え、泡立て器で均一に混ぜてなじませる。
2. 残りのメレンゲを2回に分けて加え、ゴムべらで底から大きくすくうように折り込む。
3. 全体が均一になったら混ぜるのをやめ、すぐに型へ注ぐ。

### 焼成と仕上げ

型に生地をゆっくりと流し込み、台の上に数回打ちつけて内部の大きな気泡を抜く。170℃のオーブンで約40〜45分かけてじっくり焼き、竹串を中心部に刺して何もついてこなければ焼き上がりの目安である。焼き上がったらすぐに型を逆さにして瓶の口などに挿して支え、完全に冷めるまで維持する。冷めたらシフォンナイフを外側の型の内壁に密着させ、小刻みに動かしながら一周させた後、中央の筒の外壁も同様に一周させ、最後に底板と生地の間にも刃を差し込んで切り離す。

1. 型に生地を流し込み、台の上に数回打ちつけて気泡を抜く。
2. 170℃のオーブンで約40〜45分焼く。
3. 竹串を中心部に刺して何もついてこなければ焼き上がり。
4. すぐに型を逆さにして瓶の口などに挿して支え、完全に冷めるまで維持する。
5. シフォンナイフを外側の型の内壁に密着させ、小刻みに動かしながら一周させる。
6. 中央の筒の外壁に沿っても同様に一周させる。
7. 底板と生地の間にも刃を差し込んで切り離し、底板ごと取り出す。

## 原理

シフォンケーキの軽い食感はメレンゲに含まれる無数の微細な気泡によって実現される。卵白タンパク質の主成分であるオボアルブミンは、激しく泡立てることで空気と水の界面に吸着して変性し、隣接する分子と分子間ジスルフィド結合を形成することで気泡を安定的に包む薄い膜を形成する。焼成中にこの気泡が熱膨張することで生地が大きくふくらみ、そのまま固化して軽い構造が維持される。

グラニュー糖はメレンゲの液体相の粘度を高め、泡を構成する水膜の排液速度を遅らせることで気泡の崩壊を抑制する。この効果は砂糖の濃度に比例して強くなり、低速泡立て段階での均一な気泡形成にも貢献する。また、砂糖がタンパク質の変性温度を上昇させる作用もあり、泡立て中の過変性を抑えることでなめらかな光沢のあるメレンゲが得られる。

薄力粉に含まれるデンプンは焼成中に60〜70℃で水を急速に吸収して膨潤し、アミロース分子が溶出してゲル状の連続相を形成するデンプン糊化が起こる。この糊化によってクラムに弾力のあるしっとりした組織が生まれる。同時にグルテンタンパク質（グリアジンとグルテニン）は熱変性して架橋し、膨張した気泡をその位置に固定したまま硬化することで焼き上がりの形状を維持する骨格となる。

太白ごま油などの植物油を使用する理由は、脂肪酸組成の違いにある。バターに多く含まれるパルミチン酸やステアリン酸などの飽和脂肪酸は融点が32〜35℃と高く、冷蔵温度では固体状になるためクラムの食感をかたくする。一方、植物油は不飽和脂肪酸が主体で融点が低く、冷蔵後も液体状態を保ってしっとり感を持続させる。さらに液体油はグルテン鎖を均一に覆うことができ、生地全体に柔軟性を与えて細かいクラム組織を形成させる。

乾燥卵白をメレンゲに加えるとボウル内の卵白タンパク質の総量が増加し、気泡膜の厚みと強度が増して焼成中にメレンゲが崩れにくくなる。はちみつの主成分であるフルクトースは水酸基を豊富にもち、焼成後も生地内の自由水を捕捉して保持するヒグロスコピック性を示す。この保水効果により冷却後のクラムが潤いを保ち、時間が経ってもしっとりした食感が維持されやすくなる。

ベーキングパウダーは重曹（炭酸水素ナトリウム）と酒石酸水素カリウムなどの酸性塩が混合された二段階膨張型の膨化剤であり、水分との接触で第一段の反応が起こり、加熱で第二段の熱反応が起こってそれぞれ二酸化炭素を発生する。この二段階の反応によって生地に混合した段階と焼成中の両方でガスが発生し、メレンゲの気泡と相乗して生地の膨らみを促進する。生地の膨らみはシフォンケーキの体積を決定し、食感の軽さに直接影響する。
