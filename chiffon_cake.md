# 基本のシフォンケーキ

## 基本情報

シフォンケーキは、卵白を泡立てたメレンゲを卵黄生地に折り込み、型に流して焼くことで、きめ細かくふんわりとした軽い食感を生み出すケーキである。植物油を使用するため、バターを用いるスポンジケーキよりもしっとりとした口当たりになるのが特徴で、焼成中に型の内壁を伝って高く立ち上がる独特のシルエットが生まれる仕組みをもつ。

本レシピは家庭用の分量として、17cmシフォン型1台分を基準としている。卵黄生地とメレンゲをそれぞれ別々に仕込み、丁寧に仕上げてから折り合わせることが成功の鍵であり、工程を順序よく進めることが均一な焼き上がりにつながる。風味付けにはバニラビーンズ、オレンジリキュール、キルシュを組み合わせることで、複雑で奥行きのある豊かな香りを引き出すことができる。

## 材料と道具

### 材料（17cmシフォン型1台分）

| 材料 | 分量 | 備考 |
|---|---|---|
| 薄力粉 | 120g | ふるいにかけておく |
| ベーキングパウダー | 3g | 薄力粉と合わせてふるう |
| 卵黄 | 100g | 卵黄生地用 |
| はちみつ | 10g | 卵黄生地用 |
| 太白ごま油 | 80g | 無味無臭の白ごま油 |
| 牛乳 | 120g | 全脂牛乳推奨 |
| バニラビーンズ | 1/2本 | |
| オレンジリキュール | 少量 | コアントローなど |
| キルシュ | 少量 | チェリーブランデー |
| 卵白 | 200g | メレンゲ用・冷蔵で冷やしておく |
| グラニュー糖 | 100g | メレンゲ用 |
| 乾燥卵白 | 5g | メレンゲの安定に使用 |

薄力粉120gはベーキングパウダー3gと合わせてふるいにかけておく。卵黄100gとはちみつ10gは卵黄生地用に、卵白200gとグラニュー糖100g・乾燥卵白5gはメレンゲ用にそれぞれ計量して分けておく。太白ごま油80gは無味無臭の白ごま油を選ぶことで、余分な風味が加わらず仕上がりが安定し、全脂牛乳120gを使うとクラムにコクが生まれる。

### 道具

| 道具 | 用途 |
|---|---|
| 17cmシフォン型（アルミ製） | 焼成型 |
| ハンドミキサーまたはスタンドミキサー | メレンゲの泡立て |
| 大ボウル 2個 | 卵黄生地・メレンゲの仕込み |
| 泡立て器 | 卵黄生地の混合 |
| ゴムべら | 生地の折り込み |
| 粉ふるい | 薄力粉のふるい |
| 計量スケール | 分量の計測 |
| オーブン | 焼成 |

シフォン型はアルミ製で底が外れるタイプを使用する。型には油を塗らないことが重要で、生地が型の壁をつかんで上に伸びることでシフォンケーキ特有の高さが生まれる。ハンドミキサーはメレンゲの泡立てに使用し、ゴムべらは生地を折り込む工程で活躍する。計量スケールを使って材料を正確に計量することが、安定した仕上がりを得るうえで不可欠である。

## 手順

### 準備

オーブンを170℃に予熱しておく。薄力粉とベーキングパウダーを合わせてよくふるっておく。バニラビーンズはさやに縦に切り込みを入れ、ナイフの背で種をていねいにこそいで取り出す。卵黄と卵白はあらかじめ分けておき、メレンゲ用の卵白は泡立てる直前まで冷蔵庫でしっかり冷やしておくと、泡立ちが安定しやすくなる。

1. オーブンを170℃に予熱する。
2. 薄力粉とベーキングパウダーを合わせてふるう。
3. バニラビーンズのさやに縦に切り込みを入れ、ナイフの背で種をこそいで取り出す。
4. 卵黄と卵白をあらかじめ分け、卵白は泡立てる直前まで冷蔵庫で冷やしておく。

### 卵黄生地を作る

ボウルに卵黄とはちみつを入れ、泡立て器でよく混ぜ合わせる。太白ごま油を少しずつ加えながらその都度しっかり乳化させるように混ぜ、続いて牛乳、バニラビーンズの種、オレンジリキュール、キルシュを加えてよく混ぜる。最後にふるっておいた粉類を加え、ダマがなくなるまで丁寧に均一に混ぜて滑らかな卵黄生地を仕上げる。

1. ボウルに卵黄とはちみつを入れ、泡立て器でよく混ぜ合わせる。
2. 太白ごま油を少しずつ加えながら乳化させるように混ぜる。
3. 牛乳、バニラビーンズの種、オレンジリキュール、キルシュを加えてよく混ぜる。
4. ふるっておいた粉類を加え、ダマがなくなるまで均一に混ぜる。

### メレンゲを作る

冷やした卵白にグラニュー糖と乾燥卵白を合わせたものを3回に分けて少しずつ加え、ハンドミキサーで泡立てる。最初は低速で卵白全体をていねいにほぐし、グラニュー糖の1/3量を加えたあとは中〜高速で力強く泡立て続ける。ツノが立ち、なめらかで光沢があり、やや硬めのしっかりとした状態になったらメレンゲの完成である。

1. 冷やした卵白をボウルに入れ、低速でほぐす。
2. グラニュー糖と乾燥卵白を合わせたものを3回に分けて加え、中〜高速で泡立てる。
3. ツノが立ち、なめらかで光沢があり、やや硬めの状態になったら完成。

### 生地を合わせる

卵黄生地のボウルにまずメレンゲの1/3量を加え、泡立て器で均一になるまで全体をよく混ぜてしっかりとなじませる。この工程でメレンゲの泡が多少つぶれても構わない。続いて残りのメレンゲを2回に分けて加え、ゴムべらで底から大きくすくうように折り込みながら、泡をなるべくつぶさないよう静かに丁寧に混ぜ合わせる。

1. 卵黄生地にメレンゲの1/3量を加え、泡立て器で均一に混ぜてなじませる。
2. 残りのメレンゲを2回に分けて加え、ゴムべらで底から大きくすくうように折り込む。
3. 泡をつぶさないよう、全体が均一な状態になったら混ぜるのをやめる。

### 焼成と仕上げ

型に生地をゆっくりと流し込み、台の上に数回打ちつけて内部の大きな気泡を抜く。170℃のオーブンで約40〜45分かけてじっくり焼く。竹串を中心部に刺して何もついてこなければ焼き上がりの目安である。焼き上がったらすぐに型を逆さにして粗熱を取りながら完全に冷まし、十分に冷めてから型の側面にナイフを差し込んで取り出す。

1. 型に生地を流し込み、台の上に数回打ちつけて気泡を抜く。
2. 170℃のオーブンで約40〜45分焼く。
3. 竹串を中心部に刺して何もついてこなければ焼き上がり。
4. すぐに型を逆さにして完全に冷まし、冷めてからナイフで型から取り出す。

## 原理

シフォンケーキの軽い食感は、メレンゲに含まれる無数の気泡によって生み出される。卵白タンパク質は激しく泡立てることによって変性し、疎水性の部位が空気と水の界面に吸着することで気泡を安定的に包む薄い膜を形成する。焼成中にこの気泡が熱膨張することで生地が大きくふくらみ、タンパク質と小麦粉のグルテンが熱によって固まることで、その軽い構造が焼成後も維持される。

バターではなく太白ごま油などの植物油を使う理由は、油の流動性にある。バターは常温で固体であるため、冷えると生地の食感をかたくしてしまい、保存中の口当たりにも影響が出る。一方、植物油は冷えても液体の状態を保つため、焼き上がりのしっとり感が冷蔵後も長く持続する。この特性こそが、シフォンケーキ独自のきめ細かくしっとりした食感を実現している。

乾燥卵白をメレンゲに加えるのは、卵白タンパク質の濃度を高めてメレンゲの気泡膜を強化し、泡の安定性を高めるためである。はちみつは転化糖（グルコースとフルクトース）を含み、砂糖よりも保水性が高いことから、焼き上がり後も生地のしっとり感を保つ効果がある。バニラビーンズ、オレンジリキュール、キルシュはそれぞれ異なる揮発性の芳香成分を含み、焼成後も一部が残留して複雑な風味を形成する香味素材として機能する。
